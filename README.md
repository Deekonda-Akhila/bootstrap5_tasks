# Bootstrap 5 Exercises — DN 5.0 Upskilling

## How to use
1. Open the `bootstrap5-exercises` folder in VS Code
2. Install the **Live Server** extension (by Ritwick Dey)
3. Right-click any `.html` file → **Open with Live Server**

## Folder Structure
| Folder | Topic |
|--------|-------|
| 01-setup | Setting Up Bootstrap 5 |
| 02-structure | Bootstrap Structure and Files |
| 03-grid-basics | Responsive Grid Layout |
| 04-column-layouts | Column Layouts and Grid Classes |
| 05-alignment-reordering | Alignment and Reordering |
| 06-flexbox | Responsive Flexbox Utilities |
| 07-typography | Typography |
| 08-forms | Forms |
| 09-buttons | Buttons |
| 10-navbar | Navbars and Navigation |
| 11-cards | Cards and Media Objects |
| 12-spacing | Spacing Utilities |
| 13-colors | Colors and Backgrounds |
| 14-display-visibility | Display and Visibility |
| 15-borders-shadows | Borders, Shadows, Rounded Corners |
| 16-positioning | Positioning Utilities |
| 17-icons | Bootstrap Icons |
| 18-js-plugins | Bootstrap JS Plugins |
| 19-sass-customization | Customization with Sass |

## Notes
- All exercises use Bootstrap 5 via **CDN** (no local install needed)
- Exercise 19 requires npm + sass for full Sass customization
- Bootstrap Icons CDN is included in exercises 17+
